*Indian kanoe - Malecite*
************************

Requires Unity 5.3.4 or higher.

This asset is intended for those who making an adventure, 
strategy, or action game. You can use it in indiana jones 
like setting, where player can travel from one to another 
location with some kind of transport. Or it can be a strategy 
game about indian's history period, where different tribes 
fighting among themselves and can use boats to carry some 
sort of goods for trade between tribes or for military 
purposes. It also may be used in quest games where you 
can put it in your locations to fill such environments 
like museums. And of course you can use it in some realtime 
demos as well (if you creating something like this)

/////////////////////////////////////////////
//  Indian kanoe A  - Malecite culture

It's easy to install. All stuff is categorized:

//////////////////////////////////////////
// Fbx in < FBX > folder
// Materials in < Materials >      
// Prefabs in < Prefabs >          
// Textures in < Textures >        
//////////////////////////////////////////

Other stuff, which contained in:

//////////////////////////////////////////
// < Platform for boats >
// < Scenes >
// < Standart Assets > folders 
//////////////////////////////////////////
